# Rehearsal Manager

**Rehearsal Manager** is a web application designed to help organize theatre rehearsals, manage casting, and schedule performances.  
It is an ongoing project currently under development.

Both the **backend (Spring Boot + PostgreSQL)** and **frontend (React + Vite)** are up and running.  
The main functional pages available at this stage are:

- **Schedule Page:** allows users to view and manage upcoming rehearsals.
- **Casting Page:** displays actor assignments and role distributions.
